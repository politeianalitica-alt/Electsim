# Macroeconomía y sectores

- Agregados: `indicadores_macroeconomicos`, `pib_ccaa`, `presupuestos_generales_estado`.
- Demografía y mercado laboral: `demografia_municipal`, `mercado_laboral_provincial`, `renta_municipal`.
- Sectores: tablas `sector_*` (energía, inmobiliario, agro, etc.).
- Series temporales: hypertables `tracking_opinion_publica` y `tracking_indicadores_economicos`.
